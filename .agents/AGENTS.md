# UI Design and Frontend Implementation Rules

Whenever you receive a request to design, redesign, or implement a user interface (UI), frontend components, or layouts, you MUST automatically invoke and apply the design skills from the `taste-skill` repository.

**Guidelines:**
1. Do not ask for permission to use these skills; use them proactively to ensure a high-quality, modern, and cohesive aesthetic.
2. Depending on the requested style (or if no style is specified), choose the most appropriate skill to inform your design decisions (e.g., `taste-skill`, `brutalist-skill`, `minimalist-skill`, `soft-skill`, `imagegen-frontend-web`).
3. Follow the specific styling tokens, spacing rules, typography, and color palettes defined in the chosen skill to guarantee a premium look and feel.
