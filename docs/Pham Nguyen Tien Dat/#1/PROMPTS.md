## 8. Bài học về cách viết prompt

### 8.1. Khi viết prompt, em/nhóm cần cung cấp thông tin gì để AI trả lời tốt hơn?

```text
Viết tại đây...
```
Bối cảnh bài toán (Domain): Rõ ràng là đặt sân thể thao, không phải đặt phòng khách sạn.

Công nghệ ép buộc (Tech Stack): React Functional Components, Tailwind CSS. Không nói rõ AI sẽ dùng công nghệ cũ.

Yêu cầu đầu ra (Output): Responsive, có sẵn states quản lý form.
Gợi ý:

- Mục tiêu cần đạt.
- Bối cảnh bài toán.
- Công nghệ/ngôn ngữ lập trình đang dùng.
- Input/output mong muốn.
- Ràng buộc của đề bài.
- Lỗi đang gặp.
- Format kết quả mong muốn.
- Yêu cầu AI giải thích từng bước.

### 8.2. Em/nhóm đã học được gì về cách đặt câu hỏi cho AI?

```text
Viết tại đây...
```
Kỹ thuật Meta-Prompting (Dùng AI tạo prompt cho AI). Việc nhờ Gemini viết một prompt tiếng Anh chuẩn xác để đưa vào công cụ sinh UI (Stitch) mang lại kết quả tốt hơn nhiều so với việc nhóm tự vắt óc dịch tiếng Việt sang tiếng Anh và nghĩ mô tả layout.
### 8.3. Lần sau em/nhóm sẽ cải thiện prompt như thế nào?

```text

```
Sẽ cung cấp rõ ràng hơn cấu trúc dữ liệu JSON đầu vào/đầu ra mong muốn để khi AI sinh code Frontend, biến State sẽ khớp ngay với biến Entity trên Backend Java, hạn chế việc phải rename (đổi tên) biến thủ công.
