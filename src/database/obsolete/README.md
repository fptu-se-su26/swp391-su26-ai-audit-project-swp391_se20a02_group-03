# LƯU Ý QUAN TRỌNG VỀ CÁC FILE SQL NÀY (OBSOLETE)

🚨 **CẢNH BÁO:**
Các file SQL trong thư mục này là các bản nháp thiết kế thủ công CŨ và ĐÃ LỖI THỜI.
Chúng **KHÔNG CÒN KHỚP** với Model thực tế của EF Core (Ví dụ: `EscrowWallets`, `Courts`, `BookingDetails`...).

Nếu bạn cố tình chạy các script này trong SQL Server, Entity Framework Core sẽ gặp lỗi Runtime nghiêm trọng do xung đột Schema, và hàm `EnsureCreated()` sẽ không thể hoạt động chính xác.

✅ **CÁCH TẠO DATABASE ĐÚNG CHUẨN:**
- Dự án sử dụng EF Core Code-First.
- Chỉ cần chạy lệnh `dotnet run` trong thư mục `src/backend/ProSport.API`. Hệ thống sẽ tự động gọi `EnsureCreatedAsync()` để tạo mới Database, thiết lập toàn bộ ~25 bảng chuẩn xác 100% và Seed sẵn dữ liệu.
- Nếu bạn cần script SQL thuần để nộp bài hoặc chạy trong SSMS, hãy sử dụng file **`ProSportDB_EFCore_AutoGenerated.sql`** nằm ở thư mục cha (`src/database/`), file này được tự động xuất ra (generate) từ EF Core Migrations đảm bảo chính xác tuyệt đối.
